---------------------------------------------------------
Aula 2 - Laboratório I | Entendendo a linguagem JAVA
---------------------------------------------------------

- Nesta aula deveríamos criar uma classe chamada Dado e copiar
um código pronto, disponibilizado pela professora. Neste código
haviam erros pré-programados, para que nós, alunos, conseguissemos
identificar os erros e corrigi-los.

- O próximo exercício consistiu em criar uma classe denominada 
Circulo. A classe oferece as seguintes funcionalidades:
    - alterar o raio do círculo;
    - calcular a área do círculo;
    - exibir, na tela, o raio do círculo e a área.
    
- Baseado na classe anterior, deveríamos fazer a classe Retangulo, 
com as mesmas funcionalidades da classe Circulo.
